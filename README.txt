A model and animation of a star nosed mole created only with cubes and linear algebra manipulations
open P1.html in firefox

Z = show/hide grid
0 = change camera position back to default
spacebar = change between jumpcut and animated views
U = rotate up
E = rotate down
H = head rotate right 
G = head rotate left 
T = tail rotate left
V = tail rotate right
N = star nose tentacles go out
S (3 times) = swimming motion
D = Dig motion
